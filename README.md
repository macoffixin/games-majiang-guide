# games-majiang-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容主要围绕麻将相关的指南与参考信息。页面均为独立文件，不依赖特定后端服务，可直接在浏览器中打开查看。

## 项目简介

本仓库维护了一系列关于麻将规则、策略、术语解释等内容的静态 HTML 页面。这些页面以独立、自包含的形式存在，方便离线查阅或嵌入其他环境使用。

## 目录说明

```
/
├── index.html          # 入口页面，提供页面导航
├── pages/              # 存放具体内容页面
│   ├── rules.html      # 基本规则说明
│   ├── terms.html      # 常用术语解释
│   └── strategy.html   # 策略与技巧参考
└── assets/             # 静态资源（样式、图片等）
    ├── style.css
    └── images/
```

目录结构可能会根据内容更新而调整，请以实际仓库文件为准。

## 页面归档说明

- 所有页面均为纯 HTML 文件，无需构建或安装依赖。
- 页面之间通过相对路径进行链接，保持可移植性。
- 每个页面均包含基本的样式与结构，可直接在浏览器中打开使用。
- 内容仅作为信息参考，不保证完全适用于所有规则变体或地区。

## 维护说明

- 本仓库不定期更新，新增或修正页面内容。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正。
- 页面内容力求准确，但可能存在疏漏或过时信息。
- 本仓库不提供任何形式的服务承诺或技术支持。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击 `index.html` 或 `pages/` 下的任意 HTML 文件，在浏览器中打开即可。
3. 也可通过任何静态文件服务器托管本仓库内容。
